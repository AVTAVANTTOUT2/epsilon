# Epsilon - Upload

Petite application PHP permettant l'upload de fichiers.

## Structure
- `index.php` : page principale avec formulaire d'upload et liste des fichiers
- `upload.php` : traitement de l'upload
- `header.php` : en-tête commun
- `style.css` : styles
- `uploads/` : dossier de stockage des fichiers uploadés (ignoré par git)

## Utilisation
Placer le projet dans le répertoire `htdocs` de MAMP, puis accéder via `http://localhost/uploadEPSILON`.
